# 水族館マッチング / Aquarium Maching
回答者の興味に合った水族館を提案するプログラムです。。 
独自のアルゴリズムにより、短時間かつ簡単な質問に回答いただくだけで、マッチングを行うことができます。  
ご利用は無料です。「マッチング」ボタンからアクセスできます。
